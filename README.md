# University Archive

학부 시절 진행했던 프로젝트들을 모아둔 아카이브 저장소입니다.

## 📂 프로젝트 목록 (Projects)

### 1. Cloud Computing: 영화 추천 사이트

Django 기반의 영화 추천 웹 서비스입니다. Key-Value Store(PickleDB)와 RDBMS(MariaDB)를 연동하여 효율적인 데이터 제공 구조를 설계했습니다.

- **Tech Stack**: Django, MariaDB, PickleDB
- **Key Features**: Key-Value 구조를 활용한 데이터 조회 최적화, 하이브리드 DB 아키텍처

### 2. Software Engineering: 위기상황 시뮬레이션 (Robot Pathfinding)

위험 지역 내에서 로봇이 스스로 지도를 작성하고 장애물을 회피하며 목표 지점까지 도달하는 시뮬레이션 소프트웨어입니다.

- **Tech Stack**: C#, UML, A\* Algorithm
- **Key Features**: 자율 주행 시뮬레이션, A\* 경로 탐색, UML 기반 객체지향 설계

### 3. Microprocessor: ATmega128 Rhythm Game

ATmega128 마이크로컨트롤러를 이용한 리듬 게임 컨트롤러 프로젝트입니다. SPI 통신을 통해 Master-Slave 간 데이터를 동기화합니다.

- **Tech Stack**: ATmega128 (C Language), SPI Protocol
- **Key Features**: 리듬 게임 로직, SPI 통신 기반 멀티 MCU 제어, 인터럽트 기반 입력 처리
